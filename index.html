THE UNDERSTAIR — — —

The Arcade

The Quarter is a real arcade — cabinets, pinball, a floor full of the machines you remember. There is no building yet: we're starting from an empty lot, town approvals and foundation and framing and all, with doors planned for 2027. The location gets announced the day the paperwork clears.

A World That Never Resets

Most arcade games forget you the moment you walk away. We're building one that never forgets anything. One shared fantasy world, a hundred levels deep, that every table in the building digs through together — one free run per person per day, every run moving a meter the whole room can see, every deed recorded by name in a permanent public ledger. When a level falls, it falls for everyone, forever.
